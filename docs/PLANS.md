# PLANS

## Canonical Workflow
1. Create or update a feature plan in `docs/exec-plans/active/<feature>/`.
2. Keep plan progress and task files current while implementing.
3. Move completed plans to `docs/exec-plans/completed/`.
4. Track unresolved debt in `docs/exec-plans/tech-debt-tracker.md`.

## Weekly Doc-Gardening
A weekly automation is used to detect stale docs and open fix PRs. Keep its output aligned with:
- `AGENTS.md` read order and routing
- `docs/design-docs/index.md`
- `docs/references/index.md`
- `docs/exec-plans/index.md`
